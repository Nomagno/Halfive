Half-World

the Universal racing game protocol

Half-World is a dead-simple specification that is easily expandable. It allows racing games of many different types, and even of non-racing paradigms, to communicate with each other.

To implement it in your game, you need a JSON parser, a TOML parser, and a network of any kind.
