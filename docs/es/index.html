Half-World

un protocolo universal para juegos de carreras

Half-World es una especificación sencilla y expansible, que permite a juegos de carreras muy diferentes (e incluso de otros géneros) comunicarse entre sí.

Para implementarlo en tu to juego, necesitas un lector de JSON, un lector de TOML, y una red de cualquier clase.
